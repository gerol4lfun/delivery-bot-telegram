# 📖 Простая инструкция: Создание репозиториев и запуск бота

## 🎯 Что будем делать:

1. Создадим 2 новых репозитория на GitHub
2. Загрузим калькулятор в один репозиторий
3. Загрузим бота в другой репозиторий
4. Настроим Railway для бота
5. Протестируем

---

## ЧАСТЬ 1: Создание репозиториев на GitHub

### ШАГ 1: Создать репозиторий для КАЛЬКУЛЯТОРА

1. Откройте: https://github.com
2. Нажмите **"+"** (правый верхний угол) → **"New repository"**
3. Заполните:
   - **Repository name:** `teplitsa-calculator-main`
   - **Description:** (можно оставить пустым)
   - **Visibility:** выберите **Private** или **Public**
   - **НЕ ставьте галочки** на README, .gitignore, license
4. Нажмите **"Create repository"**

**✅ Готово! Репозиторий создан**

---

### ШАГ 2: Создать репозиторий для БОТА

1. Нажмите **"+"** → **"New repository"** (или вернитесь на главную GitHub)
2. Заполните:
   - **Repository name:** `delivery-bot-telegram`
   - **Description:** (можно оставить пустым)
   - **Visibility:** выберите **Private** или **Public**
   - **НЕ ставьте галочки** на README, .gitignore, license
3. Нажмите **"Create repository"**

**✅ Готово! Репозиторий создан**

---

## ЧАСТЬ 2: Загрузка калькулятора

### ШАГ 3: Загрузить файлы калькулятора

1. Откройте репозиторий: `teplitsa-calculator-main`
2. Нажмите **"uploading an existing file"** (или перетащите файлы)
3. Откройте папку на компьютере: 
   ```
   /Users/pavelkulcinskij/Downloads/delivery-calculator-test-main
   ```
4. Выделите ВСЕ файлы и папки, КРОМЕ:
   - ❌ Папка `telegram-bot` (её загрузим отдельно)
   - ❌ Папка `backup` (если есть)
5. Перетащите файлы в окно браузера
6. Прокрутите вниз, нажмите **"Commit changes"**

**✅ Готово! Калькулятор загружен**

---

## ЧАСТЬ 3: Загрузка бота

### ШАГ 4: Загрузить файлы бота

1. Откройте репозиторий: `delivery-bot-telegram`
2. Нажмите **"uploading an existing file"**
3. Откройте папку на компьютере:
   ```
   /Users/pavelkulcinskij/Downloads/delivery-calculator-test-main/telegram-bot
   ```
4. Выделите ВСЕ файлы, КРОМЕ:
   - ❌ Файл `.env` (он секретный, не загружайте!)
5. Перетащите файлы в окно браузера
6. Прокрутите вниз, нажмите **"Commit changes"**

**✅ Готово! Бот загружен**

---

## ЧАСТЬ 4: Настройка Railway

### ШАГ 5: Подключить репозиторий к Railway

1. Откройте: https://railway.app
2. Войдите через GitHub
3. Нажмите **"+ New Project"**
4. Выберите **"Deploy from GitHub repo"**
5. Найдите репозиторий: `delivery-bot-telegram`
6. Нажмите на него

**✅ Готово! Репозиторий подключен**

---

### ШАГ 6: Добавить переменные окружения

1. В Railway нажмите на сервис (карточка с GitHub)
2. Перейдите во вкладку **"Variables"** (слева)
3. Нажмите **"+ New Variable"** и добавьте:

   Добавьте переменные (значения возьмите из своего файла **.env** в папке проекта):
   - `TELEGRAM_BOT_TOKEN`
   - `SUPABASE_URL`
   - `SUPABASE_SERVICE_ROLE_KEY`
   - при желании: `ADMIN_USER_ID` (ваш Telegram user id)

**✅ Готово! Переменные добавлены**

---

### ШАГ 7: Настроить запуск

1. Перейдите во вкладку **"Settings"** (слева)
2. Найдите **"Start Command"**
3. В поле введите: `npm start`
4. Нажмите **"Save"**

**✅ Готово! Настройки сохранены**

---

### ШАГ 8: Проверить деплой

1. Перейдите во вкладку **"Deployments"** (слева)
2. Дождитесь завершения (2-3 минуты)
3. Статус должен стать **"SUCCESS"** (зеленая галочка)

**Если статус "FAILED":**
- Откройте **"View Logs"**
- Скопируйте текст ошибки и отправьте мне

**✅ Готово! Бот задеплоен**

---

## ЧАСТЬ 5: Проверка работы

### ШАГ 9: Протестировать бота

1. Откройте Telegram
2. Найдите вашего бота
3. Отправьте: `/start`
4. Бот должен ответить!

**✅ Если бот отвечает - всё работает!**
